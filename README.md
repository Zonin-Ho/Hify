# Hify 智能问答系统

## 简介
Hify 是一个基于langchain4j的智能问答系统，它能够回答用户的问题，并且能够根据用户的问题，生成相应的答案。

## 相关技术
**后端：**
- langchain4j
- springboot3.4+
- mybatis
- mysql8.0

**前端：**
- vue3
- ant-design-vue
- vite
## 运行
**后端：**
- ==JKD17==
- 导入数据库文件
- 修改配置文件
- 运行

**前端：**
- 安装依赖(node > 18)
  > pnpm install
- 运行
  > pnpm run dev
